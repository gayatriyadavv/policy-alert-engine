# Core Feature Suite 🛠️

The platform is built around four primary functional pillars, each designed to streamline the advocacy workflow.

### 📊 Real-Time Bill Monitor
- **Interactive Grid**: A stateful data table supporting instantaneous text-based search.
- **Dynamic Categorization**: Automatic sorting into Wildlife, Marine, Climate, Forest, and Agriculture sectors.
- **Priority Badging**: Visual triage using color-coded "Critical," "High," "Medium," and "Low" statuses.

### 🚨 Smart Alerts System
- **Threshold Triggers**: Any bill crossing an impact score of 65 is automatically promoted to the "Critical" feed.
- **Proactive Notification**: Designed to alert stakeholders immediately upon ingestion of high-stakes legislation.

### 🤖 Generative AI Drafting
- **Multi-Tone Synthesis**: Drafts public comments or internal reviews in Formal, Informal, or Urgent styles.
- **Context Awareness**: The AI analyzes the specific bill title and score to generate nuanced, relevant arguments.

### 📥 Ingestion Simulation
- **Automated Scanning**: A built-in `/scan` module that replicates live data scraping for testing and demonstration purposes.
