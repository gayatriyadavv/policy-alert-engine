# Project Blueprint 📘

## 🗺️ Vision Statement
To empower ecological defenders with technological parity, ensuring that no critical legislative window is ever missed due to administrative overhead.

## 👥 Stakeholder Profiles
- **Advocacy Teams**: Primary users of the dashboard and AI drafting tools.
- **Government Affairs**: Stakeholders monitoring the global "Priority" feed.
- **Data Engineers**: Maintainers of the scraper and scoring heuristic engines.

## 💎 Project Values
- **Integrity**: Transparent and explainable scoring for all legislation.
- **Speed**: Radical reduction in the time from discovery to public response.
- **Simplicity**: No-config setup for rapid organizational deployment.
